# Commands

## Main entrypoint

- `dbibih-cli backup`
- `dbibih-cli cleanup`
- `dbibih-cli cpu`
- `dbibih-cli disk`
- `dbibih-cli update`
- `dbibih-cli docker ...`
- `dbibih-cli dockerfile ...`
- `dbibih-cli docker-monitor`
- `dbibih-cli nodejs --cleanup ...`
- `dbibih-cli python --cleanup ...`
- `dbibih-cli system --cleanup ...`

## Cleanup commands

- Node.js: `dbibih-cli nodejs --cleanup [start_dir] [ignore_file] [options]`
- Docker: `dbibih-cli docker --cleanup [options]`
- Python: `dbibih-cli python --cleanup`
- System: `dbibih-cli system --cleanup`
